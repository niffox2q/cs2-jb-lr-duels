### Модуль LR Duels для Jailbreak Core (v1.1.1)

Добавляет продвинутую систему настраиваемых дуэлей для Последнего Желания (LR) заключенных с поддержкой уникальных арен, админ-системой и полной изоляцией боя.

#### Игровые возможности:
При вызове на дуэль игрок получает доступ к меню управления, где может настроить:
* Выбор конкретного противника из живых охранников.
* Количество стартового здоровья и брони (от 25 до 100).
* Желаемое оружие для сражения (AK-47, M4A4/M4A1-S, AWP, Deagle и др.).
* **Специальные модификаторы:** Стандартный, Без прицела, Один патрон, Только ножи, 1 HP. В меню встроены интерактивные подсказки с описанием каждого режима.

**Особенности проведения дуэли:**
* **Кинематографичный старт:** После телепортации на арену игрокам дается 5-секундное окно без оружия, чтобы разбежаться и занять тактические позиции.
* **Изоляция урона:** Во время дуэли участники неуязвимы для посторонних игроков, а их выстрелы не наносят урона наблюдателям. 
* **Чистота карты:** В режиме «1 HP» выкинутые ложные гранаты автоматически удаляются после срабатывания.

---

#### Система управления аренами (Admin System):
Модуль включает внутриигровую систему для настройки локаций проведения дуэлей. 
* **Команда доступа:** `!jb_duels` (в чат) или `mm_jb_duels` (в консоль).
* **Возможности меню:**
    * **Создать позицию:** Установка точек спавна для Т и КТ на текущей карте. Координаты и угол обзора камеры сохраняются автоматически.
    * **Удалить позицию:** Удаление неактуальных арен прямо из игры.
    * **Проверить позицию:** Телепорт администратора на сохраненные координаты для проверки корректности.

> **ВАЖНЫЕ ЗАМЕТКИ:**
> 1. **Конфиг позиций изначально ПУСТ.** Модуль не содержит предустановленных точек. Администратору необходимо зайти на сервер и вручную создать арены для каждой карты через меню `!jb_duels`.
> 2. **Отсутствие арен:** Если на текущей карте не настроена ни одна локация, дуэль запустится, но игроки **останутся на своих текущих позициях**.

#### Файловая структура:
* `addons/configs/Jailbreak/lr_duels.ini` — Базовые настройки (доступ к админке, лимит времени, переключатели модификаторов).
* `addons/configs/Jailbreak/lr_duels_maps.ini` — База данных позиций. Генерируется и заполняется автоматически через админ-меню.
* `addons/translations/jailbreak_duels.phrases.txt` — Файл локализации (поддержка русского, английского, украинского языков).

### Установка:
1. Распаковать содержимое архива в папку `game/csgo/addons/`.
2. Настроить права доступа в `lr_duels.ini` (по умолчанию `@admin/root`).
3. Зайти на сервер и создать арены через `!jb_duels`.
