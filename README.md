### Модуль LR Duels для Jailbreak Core  

Добавляет продвинутую систему настраиваемых дуэлей для Последнего Желания (LR) заключенных.

При вызове на дуэль игрок получает доступ к меню управления (Hub), где может настроить:
- выбор конкретного противника из живых охранников
- количество стартового здоровья и брони (от 25 до 100)
- желаемое оружие для сражения (AK-47, M4A4/M4A1-S, AWP, Deagle и др.)
- специальные модификаторы дуэли (Стандартный, Без прицела, Один патрон, Только ножи, 1 HP)

Все базовые параметры (включая лимит времени и доступность конкретных модификаторов) берутся из конфигурационного файла `addons/configs/Jailbreak/lr_duels.ini`. Локализация модуля настраивается в `addons/translations/jailbreak_duels.phrases.txt`.

### Установка:
Распаковать в `game/csgo/addons/`
